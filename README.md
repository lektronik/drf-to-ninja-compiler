# DRF to Ninja Compiler

## Description
A tool designed to automatically parse Django Rest Framework (DRF) code (`serializers.py`, `views.py`) and convert it into Django Ninja equivalents (`schemas.py`, `api.py`).

## Features (Planned)
- Parses DRF Serializers and generates Pydantic Schemas.
- Parses DRF APIViews/ViewSets and generates Django Ninja Routers.
